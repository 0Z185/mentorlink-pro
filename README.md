MentorLink Pro is a world-class Mentorship Management System designed for high-performing corporate environments. It streamlines professional growth through AI-powered matching, automated session scheduling, and rigorous pilot program evaluation.

## Key Features

- **AI Matching Engine**: Skill-based and interest-based pairing algorithms.
- **Pilot Hub**: Dedicated administrative suite for testing and scaling mentorship cohorts.
- **Career Roadmap**: Integrated goal tracking and milestone management.
- **Automated Governance**: Real-time health monitoring of mentorship pairs.
- **Responsive UI**: A breathable, "soft-shadow" interface designed for focus.

## Tech Stack

- **Frontend**: React 19 (ES6+), Tailwind CSS
- **Icons**: Lucide React
- **Engine**: Custom TypeScript-based Matching & Governance logic
- **Data**: Mock API Service with persistence simulation

## Getting Started

1. **Prerequisites**: A modern browser.
2. **Setup**: This is a standalone frontend application. 
3. **Run**: Open `index.html` via a local development server (like VS Code Live Server).
